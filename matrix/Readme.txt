Здесь представлена библиотека для работы с матрицами.

В данный момент поддерживаются два типа матриц: 
-Общего вида (Matrix)
-Квадратная (Square_Matrix)

Для всех типов матриц реализовано:
-простейшая арифметика (+, -, *)
-универсальное сравнение (в т.ч. между разными типами)
-некоторые линейные операции над строками матриц
-решение системы линейных уравнений из этой матрицы (требуется присоединенный столбец соответствующего размера) 
-нахождение детерминанта (только для квадратной матрицы)

matrix_test и generator предназначены сугубо для тестирования библиотеки. В обычном использовании они не нужны.
