Здесь находится простейшая реализация dynamic_array.

Этот контейнер аллоцируется в процессе программы, его размер статически неопределен.
После создания контейнера реаллокации не допускаются.

Этот контейнер может быть полезен в случаях, когда содержимое обладает следующими свойствами:
-Количество объектов постоянно в течение их хранения, но непостоянно в общем.
-Необходимо фиксировать адреса каждого объекта, но допускать изменение их содержимого.
